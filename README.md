## 注意

使用 GitHub Pages 构建时：

1. 不能直接 `import` 进小说文档，而要使用 `fetch`
2. utf-8 要转换成 utf-8 with bom 格式，以防止中文乱码
3. 服务器应该是 Linux，所以会用 `\n\n` 而不是 `\r\n` 来分段
